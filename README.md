# 🛒 E-Commerce Web Application

A responsive **E-Commerce Web Application** built using **React.js**.
This project allows users to browse products, view product details, add items to a cart, and simulate a checkout process.

Products are fetched from an external API and displayed in a modern responsive interface.

---

# 🚀 Project Overview

The goal of this project is to build a modern **e-commerce frontend application** using important React concepts such as:

- Component-based architecture
- React Router navigation
- Context API for global state
- API integration
- Responsive UI design

Users can explore products, view details, manage their cart, and simulate a checkout experience.

---

# 🛠️ Technologies Used

- React.js
- JavaScript (ES6)
- React Router DOM
- Context API
- CSS3
- FakeStore API
