# 🛒 React E-Commerce Website

A fully responsive **E-commerce Web Application** built with **React.js**.  
This project allows users to browse products, view product details, add items to a cart, and simulate a checkout process.

---

## 🚀 Project Overview

The goal of this project is to build a modern e-commerce frontend using **React concepts** such as:

- Components
- React Router
- Context API
- State Management
- API Integration
- Responsive UI Design

The application fetches products from an external API and allows users to interact with a shopping cart system.

---

## 🛠️ Technologies Used

- **React.js**
- **JavaScript (ES6)**
- **React Router DOM**
- **Context API**
- **CSS3**
- **FakeStore API**

## 💻 Installation & Setup

Follow these steps to run the project locally.

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/ecommerce-app.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd ecommerce-app
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start Development Server

```bash
npm start
```

The application will open in your browser:

```
http://localhost:3000
```
